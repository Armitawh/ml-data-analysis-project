# Machine Learning: Sales Forecasting & Loan Default Prediction

## Overview
This project demonstrates **Machine Learning applications** in business data analysis, covering:

- **Linear Regression**: Predict sales based on historical data.
- **Random Forest Classification**: Predict loan defaults based on customer features.
- **Data Cleaning & Preprocessing**: Handle missing values, remove outliers, and encode categorical variables.
- **Visualization**: Compare actual vs predicted sales and evaluate feature importance.
